Text Properties

font-family: Specifies the font for an element.

Example: Courier New

font-size: Specifies the font size of text.

Example: 20px

color: Sets the color of the text.

Example: #007BFF

text-align: Specifies the horizontal alignment of text.

Example: Centered text

text-decoration: Specifies the decoration added to text.

Example: Underlined text

line-height: Specifies the line height of text.

Example: Line height of 2

letter-spacing: Sets the space between characters.

Example: 2px letter spacing

text-transform: Controls the capitalization of text.

Example: Uppercase text

Box Model

width, height: Specifies the width and height of an element.

Example: 100px by 100px

margin: Sets the margin outside of an element.

Example: 20px margin

padding: Sets the padding inside of an element.

Example: 20px padding

border: Sets the border around an element.

Example: 2px solid border

border-radius: Sets the rounded corners of an element.

Example: 10px border radius

box-shadow: Adds shadow effects around an element.

Example: Box shadow

Background

background-color: Sets the background color of an element.

Example: Gold background

background-image: Sets a background image for an element.

Example: Background image

background-position: Sets the starting position of a background image.

Example: Center background position

background-size: Specifies the size of the background image.

Example: Background size contain

background-repeat: Specifies how the background image is repeated.

Example: No repeat background
