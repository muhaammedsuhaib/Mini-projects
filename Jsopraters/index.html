JavaScript Operators Guide

Learn the various JavaScript operators and their functionalities.

1. Arithmetic Operators

Operator Description Example
+ Addition x + y
- Subtraction x - y
* Multiplication x * y
/ Division x / y
% Modulus (Remainder) x % y
++ Increment (adds 1) x++ or ++x
-- Decrement (subtracts 1) x-- or --x
** Exponentiation (power of) x ** y

2. Assignment Operators

Operator Description Example
= Assigns value x = 10
+= Adds and assigns x += 5
-= Subtracts and assigns x -= 5
*= Multiplies and assigns x *= 5
/= Divides and assigns x /= 5
%= Modulus and assigns x %= 5
**= Exponentiation and assigns x **= 2

3. Comparison Operators

Operator Description Example
== Equal to x == y
=== Strict equal (value and type) x === y
!= Not equal x != y
!== Strict not equal (value and type) x !== y
> Greater than x > y
< Less than x < y
>= Greater than or equal to x >= y
<= Less than or equal to x <= y

4. Logical Operators

Operator Description Example
&& Logical AND (both conditions true) x && y
|| Logical OR (either condition true) x || y
! Logical NOT (inverts the condition) !x

5. Bitwise Operators

Operator Description Example
& AND x & y
| OR x | y
^ XOR (exclusive OR) x ^ y
~ NOT (inverts all bits) ~x
<< Left shift x << 2
>> Right shift x >> 2
>>> Unsigned right shift x >>> 2

6. String Operators

Operator Description Example
+ Concatenation (joins strings) 'Hello ' + 'World'
+= Concatenation and assignment text += 'World'

7. Ternary (Conditional) Operator

Operator Description Example
?: Ternary operator condition ? value1 : value2

8. Type Operators

Operator Description Example
typeof Returns the type of a variable typeof x
instanceof Checks if an object is an instance of a class or constructor x instanceof Array

9. Comma Operator

Operator Description Example
, Comma separates multiple expressions x = (a, b)

10. Delete Operator

Operator Description Example
delete Deletes a property from an object delete obj.prop

11. Void Operator

Operator Description Example
void Evaluates an expression and returns undefined void(expression)

12. In Operator

Operator Description Example
in Checks if a property exists in an object "prop" in obj

13. Spread Operator (...)

Operator Description Example
... Expands elements let arr2 = [...arr1]

14. Optional Chaining Operator (?.)

Operator Description Example
?. Accesses property if it exists obj?.prop?.subprop

15. Nullish Coalescing Operator (??)

Operator Description Example
?? Returns the right-hand value if the left-hand value is null or undefined x ?? y
