Variables

var: Declares a variable that can be re-assigned.

var name = "John";

let: Declares a block-scoped variable that can be re-assigned.

let age = 30;

const: Declares a block-scoped variable that cannot be re-assigned.

const PI = 3.14;

Data Types

JavaScript has several data types:

  • String: Represents text. let greeting = "Hello";
  • Number: Represents numeric values. let score = 100;
  • Boolean: Represents true or false. let isActive = true;
  • Array: A list of values. let colors = ["red", "green", "blue"];
  • Object: A collection of key/value pairs. let person = { name: "John", age: 30 };
  • Null: Represents no value. let emptyValue = null;
  • Undefined: A variable that has been declared but not assigned a value. let notAssigned;

Operators

Arithmetic Operators:

  • Addition: let sum = a + b;
  • Subtraction: let difference = a - b;
  • Multiplication: let product = a * b;
  • Division: let quotient = a / b;

Comparison Operators:

  • Equal: if (a == b)
  • Strict Equal: if (a === b)
  • Not Equal: if (a != b)
  • Greater Than: if (a > b)

Control Structures

If Statement:

if (condition) { /* code */ }

Switch Statement:

switch (expression) {
  case value:
    // code
  break;
}

Functions

Function Declaration:

function greet() { return "Hello!"; }

Function Expression:

const greet = function() { return "Hello!"; };

Arrow Function:

const greet = () => "Hello!";

Events

Handling events in JavaScript:

document.getElementById("myButton").addEventListener("click", () => alert("Button clicked!"));

DOM Manipulation

Accessing and modifying the DOM:

document.getElementById("myElement").innerText = "New Text";

Adding a new element:

const newDiv = document.createElement("div");

AJAX

Making an AJAX request:

const xhr = new XMLHttpRequest();
xhr.open("GET", "https://api.example.com/data", true);
xhr.onload = () => { if (xhr.status === 200) console.log(xhr.responseText); };
xhr.send();

Promises

Using promises to handle asynchronous operations:

const myPromise = new Promise((resolve, reject) => {
  const success = true; // Change to false to see rejection
  if (success) {
    resolve("Operation succeeded!");
  } else {
    reject("Operation failed!");
  }
});
myPromise.then(result => console.log(result)).catch(error => console.error(error));

Async/Await

Using async/await for better readability:

const fetchData = async () => {
  const response = await fetch("https://api.example.com/data");
  const data = await response.json();
  console.log(data);
};
fetchData();

Error Handling

Using try...catch for error handling:

try {
  let result = riskyFunction();
} catch (error) {
  console.error(error);
}

Local Storage

Storing data in the browser:

localStorage.setItem("key", "value");
let value = localStorage.getItem("key");
localStorage.removeItem("key");
